# PCB SMT焊接质量检测系统

## 项目简介

本项目是一款基于Python GUI的模型训练软件，专门用于PCB SMT焊接质量检测。通过集成YOLO等先进目标检测模型，实现了从数据准备、模型训练到实时监控的完整流程。

## 功能特性

- **数据管理**：支持导入视频文件、视频片段、图片集，自动提取视频帧作为训练样本
- **模型训练**：支持多种YOLO模型（v5/v8等），提供模型参数配置界面
- **模型评估**：提供模型性能评估（mAP、精确率、召回率等），混淆矩阵生成
- **实时监控**：支持摄像头实时流处理，模型推理与结果显示，不良品标记与统计
- **人体姿态分析**（新增）：
  - 实时摄像头视频采集
  - 人体检测与位置标注
  - 17个COCO身体关节点检测与可视化
  - 骨架连接线彩色绘制
  - 关节角度实时计算（肘、肩、髋、膝）
  - 多人同时检测与统计
- **录屏功能**（新增）：
  - 实时录制监控视频
  - 支持缺陷检测和姿态分析两种模式
  - 自动保存为MP4格式
  - 显示录屏时长
  - 视频文件保存到 data/recordings 目录
- **模型类型支持**：
  - 普通目标检测模型（如 yolov8n）
  - 姿态估计模型（如 yolov8n-pose、yolov8m-pose、yolo26m-pose）
  - 实例分割模型（如 yolov8m-seg、yolo26m-seg）
  - Darknet格式模型（如 YOLOv3）
- **可视化功能**：
  - 姿态关键点绘制（17个COCO关键点）
  - 骨架连接线彩色绘制
  - 关节角度实时显示
  - 实例分割掩码绘制（彩色半透明覆盖）
  - 摄像头镜像校正
- **配置管理**：提供模型参数、训练参数、监控参数配置

## 系统要求

### 硬件

- CPU: Intel i5或更高
- GPU: NVIDIA GTX 1060或更高（推荐，用于加速训练）
- 内存: 8GB或更多
- 存储: 50GB或更多

### 软件

- 操作系统: Windows 10/11, Ubuntu 18.04+
- Python: 3.8-3.10
- CUDA: 11.3+（如果使用GPU）
